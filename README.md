# Delhi High Court Order Scraper

A modern web application to search, download, and merge court orders from the Delhi High Court website.

## Features
- **Dynamic Search**: Fetch available case types and years directly from the DHC website.
- **Captcha Handling**: Automatically handles the website's captcha mechanism.
- **Batch Download**: Select multiple orders and download them simultaneously.
- **PDF Merging**: Compiles all downloaded orders into a single, beautifully titled PDF file.
- **Premium UI**: Dark-themed, glassmorphism design with smooth animations.

## Tech Stack
- **Backend**: Node.js, Express, Axios, Cheerio (for scraping), PDF-lib (for merging).
- **Frontend**: Vanilla HTML5, CSS3, JavaScript.

## Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `npm start`.
4. Open `http://localhost:3000` in your browser.
